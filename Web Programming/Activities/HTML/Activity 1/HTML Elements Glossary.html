<!DOCTYPE html>

Defines the HTML version.

<html>

Root element of the page.

<head>

Contains document information (meta tags, title, links to styles and scripts).

<meta>

Defines metadata (encoding, description, responsiveness, etc.).

<title>

Defines the page title.

<body>

Contains all the visible content of the page.

<h1> to <h6>

Headers of different levels.

<p>

Defines paragraphs.

<a>

Creates links (href attribute defines the destination).

<img>

Displays images (src attribute defines the image path).

<ul> and <ol>

Unordered and ordered lists.

<li>

List items.

<table>

Defines a table.

<tr>, <th>, <td>

Table elements (row, header, and data).

<form>

Defines a form.

<input>

Input fields (text, password, buttons, etc.).

<button>

Creates interactive buttons.

<div>

Generic container for layout organization.

<span>

Inline container for specific styling.

<script>

JavaScript insertion.

<hr>

Horizontal line.

<br>

Line break.

<pre>

Preserves formatting of spaces and tabs.

<dl>, <dt>, <dd>

Definition lists.

<blockquote>

Block quotation.

<q>

Short quotation.

<abbr>

Abbreviation.

<address>

Contact information.

<cite>

Reference to a work title.

<bdo>

Defines text direction.

<iframe>

Embedding another web page.

<embed>

Embedding external content.

<video>

Adds videos.

<audio>

Adds audio files.

<source>

Defines alternative sources for video and audio.

<caption>

Caption for tables.

<fieldset>, <legend>

Groups form elements.

<label>

Label for form fields.

<select>, <option>, <optgroup>

Dropdown lists.

<textarea>

Text area for forms.

<datalist>

Suggestion list for input fields.

<article>, <aside>, <details>, <figure>, <figcaption>, <footer>, <header>, <main>, <mark>, <nav>, <section>, <summary>, <time>

Semantic HTML5 elements.
